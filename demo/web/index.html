Define template

To define a temaplate, select your cycle with the timeline and edit every cell with clicking it.
You can zoom into the map with your mouse's wheel or with an trackpad pinch gesture (macOS + Safari > 9.1).

To generate a random field, click the "Random" button in the top bar. This action deletes all your cycles except the first one (The one that will be generated randomly).

Generate Cycles

To generate a sepefic amount of cycles, click the "Generate" button within the bottom bar.
The template map will be the current selected cycle selected in timeline. If you already generated 10 cycles for your template and the timeline points on the last cycle, a new generate action will use this last cycle as template.
To edit a cycle just click on its cells.

Open File

To open a file, use the button "Open File" within the top bar.

You can select a binary file (You can export such a file with using the CLI version or the module of life-game) or a simple JSON file you got from exporting

You can also select a JSON file. Please keep the following in mind. If the JSON file is just containing one cycle instead, this cycles will be used. If the JSON file is containing multiple cycles, the first one will be used.
